# Tax Trail

Version: 0.1
Status: Planning

---

# Vision

Tax Trail is a local-first desktop application designed to help freelancers, contractors, consultants, and small business owners track expenses for bookkeeping and tax preparation.

The application prioritizes:

- Ownership of data
- Simplicity
- Privacy
- Offline operation
- Accountant-friendly exports

Users should be able to install the application on their own computer, record expenses, attach receipts, categorize spending, and generate reports without requiring a cloud account.

The long-term vision is to support optional cloud synchronization while preserving the ability to operate completely offline.

# Core Problems

Many small business owners currently:

- Store receipts in folders
- Track expenses in spreadsheets
- Use accounting software that is overly complex
- Lose documentation required for taxes
- Have difficulty separating personal and business expenses

The application should solve these problems while remaining easy to use.

---

# Guiding Principles

## Local First

The application must function without internet access.

## User Owns Data

Users should always have access to:

- Database backups
- Receipt files
- Exported reports

No proprietary lock-in.

## Privacy First

Financial data remains on the user's device unless explicitly synchronized.

## Simple Over Comprehensive

The goal is not to compete with QuickBooks.

The goal is to provide a focused expense tracking solution that covers the needs of most small businesses.

2. Track expenses for an entire tax year.
